What can I do with it?
  • It's just HTML and CSS
  • Each slide is a fullscreen container
  • You can use standard classes
  • Or you can add your own styles
  • Only current slide is kept visible

      <div class="type-cover">
        <span class="title">
          Presentation title
        </span>
        <span class="subtitle">Presentation subtitle</span>
      </div>
    </div>
